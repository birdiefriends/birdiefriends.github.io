Importing from GHIN
How to export from GHIN:
1. Log in at ghin.com as a club administrator
2. Go to Golfers → Roster and filter to your club
3. Click Export (CSV or Excel → save as CSV)
4. Click Import GHIN CSV above and select the file

Supported column names (the importer detects these automatically):
Name / First Name + Last Name / Player Name
Handicap Index / HI / GHIN Index / Index / HCP
