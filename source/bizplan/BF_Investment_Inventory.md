## Purpose & How This Doc Is Used

Gates 3, 4, and 5 each independently flagged open cost/effort items — competitive research, BF-E marketing, the BirdieFriends Engine's GTM-readiness gap — with no shared unit of measurement between them. This doc exists to hold them side by side.

**This is a ledger, not a plan.** Entries start as rough ranges (cost bands, timescopes) and get replaced with real numbers as dedicated research sessions produce them. Nothing here is a commitment — ranges are working estimates for comparison and sequencing, not budgets.

**Intended use:**
- Each entry names which Gate(s) it affects, so this doc doesn't duplicate the Gate Log — it complements it as a focused, cost/effort-specific view.
- Future sessions launch as **focused single-topic BZP#s** (e.g., "BZP#N - Competitive Research," "BZP#N - Engine Rearchitecture Scoping") — each one picks up one row here, does the research/debate, and returns with a real range replacing the estimate.
- Eventually, this doc (alongside Vision) is a likely source document for a **succinct public-facing presentation** — kept as a focused chunk deliberately, rather than folded into the longer Gate Log, so it stays reviewable at a glance.

**Entry format:** Item · Category · Affects Gate(s) · Cost Range · Timescope · Status · Candidate Next Session
